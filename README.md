# Bounded Predictive Machine — World-Model-First

A corrected restart of the Bounded Predictive Machine project.

This repository starts from the world-model-first architecture:

body/system state + priors + world model
→ prediction
→ signal intake
→ prediction error
→ evidence quality
→ update or no-update
→ action or no-action
→ safety
→ outcome
→ memory
→ revised model

This is not a continuation of the old manual-text-first v0.2 foundation.
